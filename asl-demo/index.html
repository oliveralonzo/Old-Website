      HARDWARE INFORMATION

      1. What is the PLTE200?

      The Powerline Network Adapter features one Ethernet port to connect a router, computer, or other network device to your Powerline network.

      2. What is the warranty period of the PLTE200?

      The warranty period is dependent on the country of purchase.  For more information about your device's warranty, click here.

      3. Does the PLTE200 have a password and how to locate the device password?

      Yes.  The device password is located on a label at the back of the device.

      4. What is the minimum requirement to support PLTS200/PLTE200?

        i.     A computer or other device with Ethernet network adapter.
       ii.     An available AC power jack.
      iii.     A password change utility that requires a computer with a CD-ROM drive running Windows XP, Vista, or Vista 64-bit with
              latest updates.

      SOFTWARE

      1. What kind of Powerline technology does the Powerline Network Adapter use?

      The PLTE200 uses a Home plug 1.0 with Turbo Powerline technology.

      2. What Operating System can be used with PLTE200?

      The PLTE200 is compatible with Windows XP and Windows Vista operating systems.

      3. What is the default network password for the PLTE200?

      The default network password is “HomePlug”.

      4. How do I access the Powerline Utility?

      Click Start > All Programs > Linksys > Powerline Utility.

      5. Can I change the network password?

      Yes.

      LEDs

      Front Panel of the Powerline Network Adapter

      1. What do the LEDs on the front panel of PLTE200 indicate?

      • Power (Blue) – The power LED lights up when the Powerline Adapter is powered ON.

      • Ethernet (Blue) – The Ethernet LED lights up blue when the Powerline Adapter is connected to a network device through the Ethernet port.  The LED flashes to indicate network activity over that port.

      • Powerline (Blue) – The Powerline LED lights up blue when the Powerline Adapter is connected to the Powerline network through the Power port.  The LED flashes to indicate network activity over that port.

      PORTS

      1. What do the ports of the PLTE200 indicate?

      • Ethernet – The Ethernet port connects to an Ethernet network device, such as a router or computer.

      • Power – The Power port connects to the Powerline end of the power cord.

      TROUBLESHOOTING

      1. None of the LEDs light up after I installed the Powerline Adapter, what should I do?

      To resolve this issue:

      Step 1:
      Disconnect the Ethernet network cable from the Powerline Adapter.

      Step 2:
      Disconnect the power cord from the Powerline Adapter and wait for five seconds.

      Step 3:
      Re-connect the Ethernet network cable to the Powerline Adapter.

      Step 4:
      Re-connect the power cord to the Powerline Adapter.

      If this solution does not work, make sure the electrical outlet is working properly.

      2. What to do when the Ethernet LED does not light up on the Powerline Adapter?

      When the Ethernet port detects a local network connection, the Ethernet LED will light up.  Follow these steps below:

      Step 1:
      Make sure the Ethernet network cable is securely connected to the Powerline Adapter.

      Step 2:
      Make sure the Ethernet network cable is securely connected to the computer or other network device.

      Step 3:
      Check the Ethernet adapter on your computer to see if the adapter is working properly.

      3. What should I do if I cannot detect all of the Powerline devices when I scan the Powerline network?

      Ensure that all devices on the Powerline network have been configured with the same Network Password.  The Network Password must be identical in order for them to communicate with each other.  If you are adding a new Powerline Adapter to a pre-existing power network with a unique Network Password, use the Add Device option on the main screen of the Powerline Utility.

      4. What do I do when I cannot connect to other computers on your Powerline network or to the Internet?

      To resolve this, take note of the following

      Step 1:
      Make sure all devices on the Powerline network have been configured with the same Network Password.

      Step 2:
      The Network Password must be identical in order for them to communicate with each other.  If you are adding a new Powerline Adapter to a pre-existing power network with a unique Network Password, use the Add Device option on the main screen of the Powerline Utility.

      Step 3:
      Verify that the IP address and TCP/IP protocol are set up correctly for all of the Powerline networked computers.

      SPECIAL FEATURES

      1. What is the transfer rate of the PLTE200?

      The transfer rate of PLTE200 is 85 Mbps at physical layer..

      2. Does PLTS200 have a web-based setup page?

      No.  It only has the Windows-based utility interface.

      3. Does PLTS200 support TCP/IP for standard routers?

      Yes.

      4. What is the internal power supply of the PLTE200?

      The internal power supply being used is 85-240V AC/50-60Hz.
